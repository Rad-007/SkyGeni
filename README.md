## Installation & Setup
- Steps to clone the repository:
  ```bash
  git clone https://github.com/Rad-007/SkyGeni.git
  cd SkyGeni
  ```

  - Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

  - Activate virtual environment:
    ```bash
    # On Windows:
    .\env\Scripts\activate
    # On macOS/Linux:
    source env/bin/activate
    ```

  - Run the script:
    ```bash
    python main.py
    ```
